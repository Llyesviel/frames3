# Pr3.ConfigAndSecurity - Управление конфигурацией и базовая безопасность

Этот проект представляет собой реализацию службы с безопасной загрузкой конфигурации и базовыми механизмами защиты (CORS, Rate Limiting, Security Headers) на базе ASP.NET Core 8.

## Инструкция по запуску

Проект поддерживает два режима работы, которые переключаются через конфигурацию:
- **Учебный** (подробные сообщения об ошибках)
- **Боевой** (минималистичные сообщения об ошибках, чтобы не раскрывать детали реализации)

Для запуска необходимо перейти в папку проекта:

```powershell
cd csharp/src/Pr3.ConfigAndSecurity
```

### Запуск в учебном режиме (по умолчанию в appsettings.json)
```powershell
dotnet run
```

### Запуск в боевом режиме (переопределение через аргументы командной строки)
```powershell
dotnet run -- --mode Боевой
```

### Запуск с переопределением через переменные окружения
Вы можете переопределить любые настройки через переменные окружения с префиксом `PR3_`.
Например, в PowerShell:
```powershell
$env:PR3_App__Mode="Боевой"
$env:PR3_App__RateLimits__ReadPerMinute="100"
dotnet run
```

## Тестирование

Для запуска тестов (включая интеграционные тесты для проверок безопасности) выполните в корне проекта `csharp`:
